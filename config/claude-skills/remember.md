# Remember

Save a fact to persistent memory so it's recalled in future conversations.

Usage: `/remember <fact to save>`

Write the fact to the appropriate memory file in `~/.claude/projects/.../memory/` using the correct type:
- `user` — about the user's role, preferences, expertise
- `feedback` — corrections or confirmed approaches
- `project` — ongoing work, decisions, deadlines
- `reference` — where to find things in external systems

Always update `MEMORY.md` index after writing the file.
